Docker的安装

Sep 21, 2018 | Docker

镜像

镜像就相当于使用安装Ubuntu时使用ISO文件，系统安装时依赖的ISO文件

1.查看docker的环境

docker image ls

2.加载docker

docker load -i 路径

3.删除镜像

docker image rm 镜像名字

4.保存

docker save -o  文件名 镜像名

容器

1.运行容器

通过 docker run [选项] 镜像名 运行就叫做 容器了

以 交互方式（退出容器就关闭了）运行

    docker run -it --name=容器的名字      镜像的名字
    退出就关闭了容器

以守护进程方式（退出了，docker还在运行）

    docker run -dit --name=容器的名字  镜像的名字
    docker exec -it 容器名/容器id

2.查看容器

docker container ls 是查询 以守护进行方式运行的容器
docker container ls --all 是查询 所有（包括运行和未运行的）的容器

3.容器的开启和关闭

docker container start/stop  容器id/容器名

4.删除容器

docker container rm 容器id/容器名
只能删除 已经停止的容器
