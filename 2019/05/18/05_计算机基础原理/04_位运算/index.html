计算机中的几种位运算

May 18, 2019 | 计算机基础

例如：
a = 3
b = 4

将十进制转为二进制（32位为例）

a： 00000000 00000000 00000000 00000011

b： 00000000 00000000 00000000 00000100

1. 位与

规律：有0则0;  bit位之间0和1进行比较，可以看作是true和false之间and的比较
        a : 00000000 00000000 00000000 00000011
        b : 00000000 00000000 00000000 00000100
结果: a&b = 00000000 00000000 00000000 00000000 = 0(十进制)

2. 位或

规律：有1则1;  bit位之间0和1进行比较，可以看作是true和false之间or的比较
        a : 00000000 00000000 00000000 00000011
        b : 00000000 00000000 00000000 00000100
结果：a|b = 00000000 00000000 00000000 00000111 = 7 (十进制)

3. 位异或

规律： 相同则0，不同则1
        a : 00000000 00000000 00000000 00000011
        b : 00000000 00000000 00000000 00000100
结果：a^b = 00000000 00000000 00000000 00000111 = 7 （十进制）

4. 按位取反

规律：1变0，0变1
例如：    c = -4 = 10000000 00000000 00000000 00000100(原)
按位取反结果： ~c = 11111111 11111111 11111111 11111011(反) 得到反码
按位取反后低位+1 得到补码

5. 计算内部的运算方式

计算机中所有的运算都是以补码的形式进行运算的，例如在程序中定义某变量a=5，实际是经过以下过程

变量a --> 转为二进制原码 --> 转为二进制反码 --> 转为二进制补码 --> 然后开始进行运算
