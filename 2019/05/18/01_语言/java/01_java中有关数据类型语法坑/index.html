Java语言中有关数据类型的坑

May 18, 2019

简介

Java语言是强类型语言，对于每一种数据都定义了明确的具体数据类型，在内存总分配了不同大小的内存空间

数据类型分为以下几种类型:

- 基本数据类型
    - 数值型
        - 整数类型(byte, short, int, long)
        - 浮点类型(float, double)
    - 字符型(char)
    - 布尔型(boolean)
- 引用数据类型
    - 类(class)
    - 接口型(interface)
    - 数组([])
类型 所占空间 表输范围
byte 1字节 -2^7 ~ 2^7-1
short 2字节 -2^15 ~ 2^15-1
int 4字节 -2^31 ~ 2^31-1
long 8字节 -2^63 ~ 2^63-1

总结

  • Java语言的整型常量默认为int型，声明long型常量 尾部必须加‘ l ’或‘ L ’否则会出错
  • 类型转化从小向大转换不会报错，反之报错丢失精度，因为字节数不同，会被截取
  • 常量可以直接定义数据类型，变量会进行隐式转换类型为int后运算
  • 类型转化的顺序：(byte,short,char)-—>int-—>long-—>float-—>double
  • 定义浮点型小数后加f或者F
  • 强制类型转换的格式 byte c = (byte) (a + b)
  • 将两个较小的常量相加，赋值给byte类型不会报错，先常量相加再判断是否在要赋值的变量类型范围内
  • 强制数据类型转换数据溢出后会从最小值开始累加截取的数值，地球是圆的，走了一圈后又从起点开始走
  • 单个字符会被当作字符串打印，但一个字符和整数进行计算的过程就是：’a’ + 1 字符会先转换为a对应asiic码上的数值再加1，所以得到的还是数值，如果字符串在前就是来字符串拼接
