hadoop环境搭建（单机自用）

Jul 13, 2019 | Hadoop

安装java环境

略

下载hadoop

1.下载hadoop安装包，使用国内的镜像快一点

wget http://mirror.bit.edu.cn/apache/hadoop/common/hadoop-2.9.0/hadoop-2.9.0.tar.gz

2.解压文件

tar -zxvf hadoop-2.9.0.tar.gz -C /usr/local/

3.重命名

cd /usr/local/
mv hadoop-2.9.0/ hadoop

4.vim /etc/profile 修改环境变量 添加以下变量

export HADOOP_HOME=/usr/local/hadoop 
export HADOOP_MAPRED_HOME=$HADOOP_HOME 
export HADOOP_COMMON_HOME=$HADOOP_HOME 
export HADOOP_HDFS_HOME=$HADOOP_HOME 
export YARN_HOME=$HADOOP_HOME
export HADOOP_COMMON_LIB_NATIVE_DIR=$HADOOP_HOME/lib/native 
export PATH=$PATH:$HADOOP_HOME/sbin:$HADOOP_HOME/bin

5.source /etc/profile 使配置文件生效

6.cd /usr/local/hadoop/etc/hadoop 修改hadoop的xml配置文件

  • vim hadoop-env.sh文件中添加java环境变量

  • vim hdfs-site.xml 在 添加如下内容

  • vim core-site.xml 在 添加如下内容

7.Hadoop启动

第一次启动Hadoop需要初始化
cd /usr/local/hadoop/bin 目录下输入

./hadoop namenode -format

然后 切换到/usr/local/hadoop/sbin 目录下输入

start-dfs.sh
start-yarn.sh

8.可以输入 jps 查看是否成功启动

在浏览器输入:
http://localhost:8088 查看hadoop集群

输入:
http://localhost:50070 查看文件服务器
